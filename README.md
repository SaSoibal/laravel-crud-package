# Laravel CRUD Package
## Required PHP version:
```php
  "php": "^8.1",
```
## Installation
```sh
composer require sasoibal/crud:dev-main
```
## Next install database migration Commend:
```php
php artisan migrate
```
### Run Project:
```sh
php artisan serve
```
### Preview crud generator go to this url:
```php
http://127.0.0.1:8000/crud/index
```
